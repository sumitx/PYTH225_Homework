# Read Me

## Summary
This project aims to analyze trends in historical video games sales within the past several years. We'll be looking specifically at the growth or decline in specific Genres of video games during this period.

The analysis will try to make sense of the historical data, trends through bar graphs, and data outliers

## Components

This project relies on a CSV file called "vgsales.csv". This data was pulled from Kaggle https://www.kaggle.com/mohalim/video-games-sales

The actual analysis and project can be found in Sumit_Pal_Final_Project.ipynb

This file can be opened up with Colab and run from there
